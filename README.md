# Umgalelo – Digital Burial Society Management System

## Project Overview
This project is a digital system for managing burial societies.  
It will help track members, contributions, and records.

## Branch Structure
- main → Final version
- test → testing branch
- each member should create their own branch

## Rules
- Do not push directly to **main**
- Work in your own branch
- Merge to **test** when your feature works
